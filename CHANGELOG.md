# Changelog

All notable changes to this project will be documented in this file.

## 0.42.2

- Outline: Added support for Beamer `\begin{frame}` – frame titles are shown and frames are nested dynamically based on their structural context (`\section`, `\subsection`, etc.)

## 0.42.1

- Added outline support for exam class environments (such as \question and \part)  

## 0.42.0

- Initial release of TeXify
- Added LaTeX outline and document symbols
- Added compile button and configurable script execution
- Added five custom actions with optional keybindings
- Added optional installation flow for the companion LaTeX extension
- Added localization support for English, German, and French
